# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, by
reference. The full text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: we want a friendly, harassment-free environment for everyone
participating in this project — issues, pull requests, discussions,
review, chat, anywhere associated with polylocale.

## Reporting

To report a Code of Conduct concern, contact the maintainer privately:

- **Email:** `jendrzejczak.mariusz@gmail.com` with subject
  `[polylocale CoC]`

Reports are confidential. The maintainer will read them within a few
business days and respond. This is a one-person project, so response
times are best-effort.

## Enforcement

The maintainer is responsible for clarifying and enforcing this Code
of Conduct, and may take any action they deem appropriate, including
warning, temporary, or permanent removal from project spaces.
